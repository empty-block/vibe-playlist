Design Specifications

  • Style: Napster toolbar aesthetic - simple black outlines with light pastel fills
  • Base Size: 32x32px (scales to 16px, 24px, 48px)
  • Stroke Width: 2px for main elements, 1.5px for details
  • Colors: Light blue (#b8d4ff), light pink (#ffb8c0), light green (#d4ffb8) with black (#000) outlines
  • Design Principle: Minimal detail, maximum clarity - exactly like original Napster toolbar icons
  • Philosophy: Each icon uses only essential shapes; readable even at smallest sizes

Implementation Notes

  • Channels: Double musical note (beamed eighth notes) + radio wave curves = broadcasting music
  • Activity: Stylized flame shape = hot/trending content (matches Napster's "Hot List")
  • Profile: Head circle + shoulder arc = user identity (cleanest possible person icon)
  • Scalability: All icons use same 32x32 viewBox for consistent scaling
  • Accessibility: High contrast black outlines ensure visibility on any background
