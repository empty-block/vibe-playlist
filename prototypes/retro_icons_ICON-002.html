Design Notes - 90s Application Icon Style

  • Color Palette: Vibrant primary colors (magenta #d946ef, cyan #06b6d4, yellow #fbbf24, green #22c55e)
  • Shading Technique: Simple 3-level shading (highlight, base, shadow) mimicking early 3D rendering
  • Black Outlines: 1.5-2.5px solid black strokes for clear definition and readability
  • Icon Metaphors: Literal, recognizable objects (boombox = music/channels, graph = activity, ID card = profile)
  • Playful Details: Antennas on boombox, smiley face on ID card, upward arrow on graph
  • Multiple Colors: Each icon uses 4-5 distinct colors for visual richness
  • Scalability: Designed at 64x64px but scales cleanly to 32x32px for navigation bars
  • Historical Reference: Inspired by Winamp (boombox aesthetic), Windows 98 system icons (simple geometric shapes), and ICQ/AIM era (friendly, approachable)
