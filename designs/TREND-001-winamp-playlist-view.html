TREND-001: Classic Winamp Playlist View

Design Philosophy

  • Immediate Playability: Large cyan play buttons make every track instantly accessible
  • LCD Aesthetic: Green monospace text (#00FF00) on black matches the player's look
  • Information Density: 6 tracks visible without scrolling on most devices
  • Visual Hierarchy: Primary (tracks) → Secondary (artists) → Tertiary (contributors)

Key Features

  • Time-Weighted Display: Filter tabs (1H, 6H, 24H, 7D) for different time windows
  • Trending Indicators: Fire badge (🔥 HOT) for top performers
  • Social Proof: Share counts and recency timestamps
  • Win95 Chrome: Title bar, borders, status bar all authentic
  • Retro Scrollbar: Pixelated Win95-style with arrow buttons

Component Specifications

  • Track Items: 64px height (56px mobile), 48×48px thumbnails, 48×48px play button
  • Artist Cards: 3-column grid (2-column mobile), 48×48px avatars
  • Contributors: Compact list, 32×32px avatars, rank numbers
  • Colors: LCD green (#00FF00), cyan (#00FFFF), yellow (#FFFF00), magenta (#FF00FF)
  • Spacing: 8px base unit throughout (4px, 8px, 12px, 16px...)

Interaction Details

  • Hover: Track items get subtle green glow background (#001a00)
  • Play Button Hover: Cyan → Yellow with glow effect
  • Active Filter: Inverted Win95 button (depressed look)
  • Touch Targets: Minimum 44px for all interactive elements

Empty State

  • Centered lightning bolt icon (48px)
  • Green text: "NO TRENDING ACTIVITY"
  • Cyan subtext: "Share some tracks to heat things up!"
  • Matches LCD panel styling

Responsive Behavior

  • <640px: Reduce track height to 56px, artist grid to 2 columns
  • <360px: Further compression to 52px tracks, smaller fonts
  • Scrolling: Vertical only, retro scrollbar always visible

Technical Notes

  • Font: 'Courier New' monospace for LCD, 'MS Sans Serif' for Win95 chrome
  • Border style: 2px solid with light/dark inset/outset effect
  • Status indicator: Animated green dot pulse (2s ease-in-out)
  • Fixed bottom spacing: 80px for nav + player (matches existing layout)
  • Max width: 480px (typical mini-app frame)
