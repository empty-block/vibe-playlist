Windows XP Design Specifications

  • Size: 32×32px (standard XP icon dimensions)
  • Style: Slight gradients with subtle highlights, mostly flat design
  • Colors: Authentic XP palette - blues (#0078D7, #3399FF), greens (#00CC00), oranges (#FF9900, #FFA726), grays (#808080)
  • Aesthetic: Clean, functional, professional - exactly as Microsoft shipped in 2001
  • Details: Simple shapes, clear silhouettes, minimal embellishment
  • Navigation: Works in both light (XP Classic) and dark themes
  • Inspiration: Windows Media Player, My Music folder, My Documents, User Accounts
