Design Notes - v5.3

Key Changes from v5:
  • Source badges now use icons instead of text labels
  • Spotify: Green circle with "â™«" symbol
  • YouTube: Red circle with "â–¶" symbol
  • Icons positioned at bottom-right of thumbnail (18px circles)
  • Tooltip on hover reveals source name for accessibility
  • Scale animation on hover for subtle interactivity
Neon Play Button:
  • Cyberpunk/terminal inspired design
  • Transparent background with green border
  • Green glow effect (box-shadow)
  • Intensified glow on hover
  • Inline play icon (â–¶) before text
  • Minimal and functional aesthetic
Benefits of Icon Approach:
  • Takes up less visual space than text badges
  • More subtle and elegant
  • Colorful badges create instant recognition
  • Better for international users (symbols > words)
  • Cleaner card design overall
  • Works well at small sizes
Maintained from v5:
  • 48x48px thumbnails with gradients
  • Yellow-bordered comment boxes
  • Clean card layout and spacing
  • Hover states on interactive elements
  • Responsive stat display
Technical Details:
  • Source icons: 18px circles, absolute positioned
  • Play button: 1px border, 5-15px glow on hover
  • Icon symbols: Unicode characters for compatibility
  • Spotify green: #1db954 (brand color)
  • YouTube red: #ff0000 (brand color)
  • Neon green: #00ff41 (terminal aesthetic)
