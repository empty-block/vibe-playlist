Design Notes

Philosophy: These icons follow the iTunes/iPod Classic design language - simple geometric shapes that a child could draw, but perfectly balanced and instantly recognizable.

Channels: Musical note represents content/music discovery. The eighth note is the simplest musical symbol everyone recognizes.

Activity: Play triangle represents action and movement - perfect for an activity feed. Same shape used in the iPod play button.

Profile: Person silhouette is universal and timeless. Simple circle head + body shape.

Color Usage: Default state uses #666 gray. Active state uses #0066cc blue (classic iTunes blue). Optional colored circles for visual interest (iPod menu style).

Size: 24×24px base size, perfectly crisp at any scale. Circular background variant is 32×32px.
