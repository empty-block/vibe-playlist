Design Notes

  • CHANNELS: Classic Windows folder icon (yellow/orange) with simple music note. Like WinAmp playlist folder.
  • ACTIVITY: Classic lightning bolt icon - simple angular zigzag shape. Yellow/gold like Windows power/warning icons. Represents speed, energy, and activity.
  • PROFILE: Generic user icon - circle head + shoulders. Light blue like Windows XP default user icon.
  • Limited color palette: ~3-4 colors per icon (folder: 3 yellows, activity: 2 golds, profile: 3 blues)
  • Black outlines (0.5px stroke) for clarity at small sizes
  • Simple geometric shapes - no fancy paths or gradients
  • Designed to work at 16x16px (standard toolbar) and 24x24px (larger UI)
  • Style: Windows XP/2000 era - functional, not fancy

⚠️ DESIGN PHILOSOPHY

These icons are intentionally boring and functional. They look like they were designed by a Windows developer in 2001 who had 10 minutes before lunch. No artistic flair. No clever metaphors. Just simple, recognizable shapes that clearly communicate their function.

This is feature, not a bug. Real software from this era prioritized clarity and functionality over aesthetics.
