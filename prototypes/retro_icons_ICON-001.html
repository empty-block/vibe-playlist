Classic 90s Software Toolbar Icons

Design Philosophy:
Authentic Windows 95/98 era toolbar icons inspired by Napster, Winamp, ICQ, and AIM. Simple, functional shapes with clean outlines and subtle 3D shading. Each icon uses 2-3 shades of color to create depth while maintaining instant recognizability.
